Tasker-Profile-Credentials

Purpose: This is a personal, single-user desktop application that connects the Hermes Agent (Tasker profile) to Google Workspace. It enables the owner to automate personal tasks — reading and sending Gmail, managing Calendar events, accessing Drive files, and working with Sheets and Docs — entirely from a local terminal. No data leaves the owner's machine except standard Google API calls to the owner's own account.

Privacy Policy

Effective date: July 24, 2026

1. What this app does

This is a personal, single-user desktop application (Tasker-Profile-Credentials) that accesses Google Workspace APIs — Gmail, Calendar, Drive, Contacts, Sheets, and Docs — solely for the owner's own use.

2. Data collection

This application collects no user data. It does not:

  • Store, log, or transmit your emails, calendar events, contacts, or files to any third party
  • Use analytics, cookies, tracking, or telemetry of any kind
  • Share any information with advertisers or data brokers
  • Run as a web service accessible to the public

All Google data accessed through the API stays on your own machine. OAuth tokens are stored locally and never leave your device.

3. How data is used

Google API access is used exclusively for the features the owner has configured, such as:

  • Reading and sending Gmail messages
  • Creating and managing Calendar events
  • Uploading and downloading Drive files
  • Accessing Contacts for address lookups
  • Reading and updating Sheets and Docs

All processing happens locally. No data is sent anywhere except back to Google's own APIs.

4. Third-party sharing

No data is shared with any third party. Period.

5. Security

OAuth credentials and refresh tokens are stored in a local file on your device with standard file-system permissions. You can revoke access at any time via Google Account Permissions.

6. Contact

This is a personal-use app with no organization behind it. If you have questions, contact the owner through the repository where this page is hosted.

7. Changes

Any changes to this policy will be reflected on this page.
